Usage

1. If you already have a Create React App project, drop these files into your project:

   - Replace `src/App.js` and `src/index.js` with the files provided.
   - Add `src/components/*` and `src/styles.css`.
   - Replace `public/index.html` with the provided file or merge the root `div`.

2. Install dependencies and run (if using CRA):

```powershell
npm install
npm start
```
